# VikaTable

基于 react-window  Grid 构建的表格组件。

react-window 是泛用性的虚拟滚动套件，vikatable 基于其中的 Grid 构建，通过扩展 Grid 的 props 参数，非侵入式支持更多的表格特性, 使其更方便构建表格。

## Feature
- [x] 使用 typescript 重写（react-window 使用 flow）
- [x] 表头 & 冻结列 （sticky）
- [x] 统计栏
- [ ] 通用的表格传参模式
- [ ] 分组

## Install
```
yarn add @vikadata/vikatable
```
